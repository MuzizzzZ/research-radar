Config Studio

Set up topics and baseline papers with a search-first workflow designed for non-technical users.

Quick Start

You only need to decide what to track. The page can search papers and fill most identifiers for you.

  1. Create one or more topics with a name and a few search phrases.
  2. Search for a baseline paper by title, DOI, arXiv id, or author keywords.
  3. Pick a search result to import it as a baseline automatically.
  4. When everything looks right, download YAML or commit the config back to GitHub in the advanced section.
