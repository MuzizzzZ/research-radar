About

research-radar is an incremental literature tracking system designed for repository-only deployment.

Architecture

GitHub Actions + repository-backed JSON/Markdown state + GitHub Pages static site

The project avoids long-running servers, online databases, and custom backend hosting.

Data Sources

  • OpenAlex
  • Semantic Scholar

Limitations

  • The system relies on public metadata availability and API uptime.
  • Title-based fallback resolution can miss or over-match some baseline papers.
  • Rule-based relevance scoring is intentionally transparent but not exhaustive.
